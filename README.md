# 表格生成器 - 轻茗yo (ExcelsMaker)

## 📖 简介

这是一个为游戏《明日方舟》方便录关而制作的小工具
第一次搞这种，有很多不完善的地方
是我很久之前搞的项目，现在才想起来上传。联网功能可能由于哔哩哔哩token问题而无法使用

## ✨ 主要功能

* **📝 自动生成关卡列表**

  * 支持自定义关卡主题（如 UR, WD 等）。
  * 可灵活配置普通关卡、EX 关卡及 S 关卡的数量。
* **🔄 数据导入与导出**

  * **导入**：支持读取现有的 `.xlsx` 或 `.csv` 文件，自动识别表头并合并数据。
  * **导出**：将整理好的数据一键导出为格式规范的 Excel 文件（自动按时间戳命名），保存在 `excels/` 目录下。
* **🌐 视频信息自动补全**

  * 集成爬虫功能，支持 Bilibili 视频链接或 BV 号。
  * **自动获取**：能够自动联网抓取视频的 阵容，人数，up主 等信息，无需手动复制粘贴。
* **🎨  GUI 界面**

  * **工具主题**：早苗色调，致敬传奇四星队攻略者。
  * **便捷交互**：支持双击打开链接、选中行快速编辑、一键清空等操作。

## 📖 使用指南

    **启动项目**：打开项目根目录“点我启动项目”文件

1. **初始化表格**：

   * 在顶部“参数设置”栏，输入本期活动的主题（例如 `UR`）。
   * 设置各类型关卡的数量。
   * 点击 **“生成关卡列表”**。
2. **录入数据**：

   * **手动录入**：点击表格中的任意一行，在下方编辑区填入“链接”、“阵容”、“备注”等信息，点击“保存修改”。
   * **文件导入**：如果有现成的部分数据，点击 **“导入 Excel/CSV”** 进行合并。
3. **联网补全**：

   * 如果你不想自己输入视频信息，可以试试联网补全功能
   * 确保“链接”列已填入 B 站视频链接或 BV 号。
   * 点击底部 **“联网补全信息”**。
   * 程序将自动运行，补全 UP 主和人数、阵容等信息。
4. **生成结果**：

   * 确认所有信息无误后，点击 **“生成最终 Excel”**。
   * 程序会提示文件保存路径（通常在 `excels/` 文件夹中）。

## ⚠️ 注意事项

* 联网补全功能需要识图，获取视频阵容信息，速度会很慢
* 联网功能目前还有bug，包括但不限于：助战干员识别不了，干员识别不全等问题
* 请手动检查一下，确保所有信息都正确
